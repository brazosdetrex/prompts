TAREA: Genera tarjetas Anki sobre el contenido recibido para facilitar memorización.

2 PASOS:
1. Analiza el contenido a nivel general: ideas, conceptos, términos, procedimientos, fórmulas, usos clave.
2. ESPERA confirmación antes de generar.

CONTENIDO DE TARJETAS
- Aprendizaje "bajo nivel": ideas aisladas, datos factuales, definiciones, nombres, datos, fechas (cosas que memorizar individualmente y no funcionan bien en conjunto)
- Tarjetas atómicas, un solo concepto, autocontenidas, **sin depender referencias a contenido externo**.
- Aborda conceptos complejos desde **distintos ángulos** -> facilitar comprender significado profundo.
- No escatimes en cantidad de tarjetas

FORMATO DE TARJETAS:
Tipo tarjeta ANKI: Básicas
- Enunciado Pregunta: 1-2 frases 
- Respuesta: 1-3 frases
Divide en varias tarjetas las listas >4 elementos


FORMATO DE SALIDA:
- Archivo: (nombre_tema).txt
- Separador: TAB entre pregunta y respuesta
- Codificación: UTF-8
- HTML para formato: Ej.<b>negrita</b>, <br> salto línea
- NO uses Markdown
