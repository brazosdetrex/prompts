Prompt para generar flashcards
Para obtener una mejor respuesta del LLM con este prompt, seguir los siguientes passos.

Curar archivo de entrada (ideal):
Formato
- Texto plano (.txt o .md)
- PDF con OCR
Contenido
- Fragmentar los apuntes en bloques mas pequeños de < 5000 palabras para obtener mejor resultado

Contexto:
Editar prompt Añadir sección de "Contexto" en la que expliques de que van los apuntes (carrera/curso, asignatura, bloques y temas) y si quieres que siga un enfoque en concreto
